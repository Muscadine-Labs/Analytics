# Muscadine Analytics

Next.js dashboard for Muscadine vaults on Morpho (Base).

## Quick Start

```bash
npm install
```

Create `.env.local` with `ALCHEMY_API_KEY` or `COINBASE_CDP_API_KEY`, then:

```bash
npm run dev
```

Open http://localhost:3000

## Scripts

- `npm run dev` – Development
- `npm run build` – Production build
- `npm run lint` – Lint
- `npm test` – Tests
